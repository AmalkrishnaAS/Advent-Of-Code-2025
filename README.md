## Structure

Each day's solution is in its own directory (`01/`, `02/`, etc.) containing:
- `solution.cpp` - The C++ solution
- `input.txt` - Puzzle input
- `sample.txt` - Sample input from the problem description
- `a.out` - Compiled binary (ignored by git)

## Building and Running

To build and run a solution:
```bash
g++ solution.cpp -o solution
./solution < input.txt
```
